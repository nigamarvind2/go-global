# Go Global EdTech

Go Global EdTech - Expert guidance for studying abroad. MBBS overseas, international courses, and more.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How to run locally

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
